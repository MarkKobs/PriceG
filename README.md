# GuessPrice
初高中教学用，猜价格，帮助学生了解二分法思想。

## 版本1.0
### 功能：
1. 判断价格高低
2. 设置价格
3. 重新开始（随机数）

## 版本3.0

### 功能介绍：
1. 商品列表展示（商品图片、商品名称）
2. 添加商品（商品图片、商品名称、商品价格）
3. 选某个商品进行猜价格
4. 智能化次数限制（根据二分策略）（算法：S=规模，x为次数，2^x-1<S，2^x>=S）
5. 高低提示
6.正确范围提示（绿色坐标轴显示）
